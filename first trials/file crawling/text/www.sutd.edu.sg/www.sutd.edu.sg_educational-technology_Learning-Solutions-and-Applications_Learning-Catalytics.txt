Learning Catalytics
Learning Catalytics (LC) is an interactive student response system that encourages interactive instant classroom feedback and team-based learning by using students' electronic devices. Faculty and students will be given a licensed account to access LC.

To sign up, please write to us at edi_admin@sutd.edu.sg.
Learning Catalytics Instructor Guide  Learning Catalytics User Guide (Student) Features


Get real-time feedback in the classroom


Use dynamic analytics to impact instructions during class

Reduce preparation time by sharing content with instructors in SUTD and around the world
